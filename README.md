# BancoDigitalPOO_Estudo
Este projeto faz parte de um desafio da DIO que simula um banco digital, com o objetivo de reforçar os conhecimentos em Java e Programação Orientada a Objetos. A aplicação aplica na prática os pilares da POO, como abstração, encapsulamento, herança e polimorfismo, em um contexto bancário.
